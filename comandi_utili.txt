Vedere info tabella:
    DESCRIBE [nome_tabella]

Inserire dati:
    INSERT INTO [tabella] ([colonna_1], [colonna_2]) VALUES ([valore_1], [valore_2])
    
Eliminare tutti i dati di una tabella:
    TRUNCATE TABLE [nome_tabella];

Modificare una tupla:
    UPDATE nome_tabella SET colonna = valore WHERE condizione;

Modificare una colonna:
    ALTER TABLE nome_tabella MODIFY nome_colonna NUOVO_TIPO;

Aggiungere una colonna:
    ALTER TABLE nome_tabella ADD COLUMN nome_colonna TIPO [vincoli];

    (Se in tabella esistente aggiungere alla fine DEFAULT valore affinch√® le tuple esistenti si riempiano)

Eliminare una riga:
    DELETE FROM nome_tabella WHERE condizione;
